# ⚡ Flash – Wireless Data Sharing App

**Flash** is a fast and lightweight wireless file sharing server built using **Flask** that allows you to transfer files between devices over the same Wi-Fi network using a web browser.

No cables. No cloud upload. No third-party apps.  
Just open the browser and start sharing instantly.
